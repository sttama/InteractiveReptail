# Интерактивная рептилия
# Описание
Этот проект создан для развлекательных целей и не имеет полезных функций (разве что, пользователь может узнать что-то новое, при изучении кода). Проект представляет собой нарисованную "рептилию", которая следует за курсором.
# Инструкция для запуска
Для начала проверьте установлена ли у вас Java (JRE), чтобы запускать программы на java.  
Для этого введите в командную строку это:  
  
`java --version` 
  
Вы должны получить версию Java и что-то еще:  
  
`java version "21.0.7" 2025-04-15 LTS`  

Если у вас не установлена Java или выводится ошибка, то скачайте ее:  [как скачать Java](https://www.java.com/ru/download/help/download_options.html) ;  
Если у вас есть Java, то можно приступить к запуску.
___
## В Windows

**1 способ**  (через исполняемый файл)

- Скачайте папку InteractiveReptail_jar
- Щелкните по файлу lounch_Windows.bat
- Готово, можете наслаждаться

**2 способ**  (через команду в терминале)

- Скачайте папку InteractiveReptail_jar
- Откройте терминал и перейдите в скачанную папку: `cd путь_к_папке/InteractiveReptail_jar`
- Далее вводите: `java -jar InteractiveReptail.jar`
- Нажимаете enter и кайфуете

**3 способ**  (через IDE (например, IntelliJ IDEA))

- Скачайте весь репозиторий
- Откройте этот репозиторий/проект в IDE (например, IntelliJ IDEA)
- Запустите файл/класс Starter.java (или просто Starter)
- Все готово, пользуйтесь

## В Linux или MacOS

**1 способ**  (через исполняемый файл)
