# **Interactive Reptile**:crocodile:
# Description :eyes:
This project was created for entertainment purposes and has no useful functions (except that the user can learn something new while studying the code). The project is a painted "reptile" that follows the cursor.

# Instructions for launching :rocket:
First, check if you have Java (JRE) :coffee: installed to run java programs.  
To do this, type this into the command prompt:  
  
`java -version` 
  
You have to get the Java version and something else:
  
`java version "21.0.7" 2025-04-15 LTS`  

If you do not have Java installed or an error is displayed, then download it:  [how to download Java](https://www.java.com/en/download/help/download_options.html) ;  
If you have Java, then you can start the startup.
___
## On Windows :window:

**1 method**  (via an executable file)

- Download the folder InteractiveReptail_jar
- Click on the file lounch_Windows.bat
- That's it, you can enjoy :checkered_flag:

**2 method**  (using a command in the terminal)

- Download the folder InteractiveReptail_jar
- Open the terminal and navigate to the downloaded folder: `cd путь_к_папке/InteractiveReptail_jar`
- Next, enter: `java -jar InteractiveReptail.jar`
- the end :checkered_flag:

**3 method**  (through an IDE (for example, IntelliJ IDEA))

- Download the entire repository
- Open this repository/project in the IDE (for example, IntelliJ IDEA)
- Run the file/class Starter.java (or just a Starter)
- Everything is ready, use it :checkered_flag:

  ___

**1 method**  (via an executable file in the terminal)

# Инструкция для запуска :rocket:
Для начала проверьте установлена ли у вас Java (JRE) :coffee:, чтобы запускать программы на java.  
Для этого введите в командную строку это:  
  
`java -version` 
  
Вы должны получить версию Java и что-то еще:  
  
`java version "21.0.7" 2025-04-15 LTS`  

Если у вас не установлена Java или выводится ошибка, то скачайте ее:  [как скачать Java](https://www.java.com/ru/download/help/download_options.html) ;  
Если у вас есть Java, то можно приступить к запуску.
